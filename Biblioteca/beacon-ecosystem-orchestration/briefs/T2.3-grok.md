## Task

Add a native note source so the Shaper can be played directly from a MIDI keyboard without NaturalHarmony or beacon broadcast running.

1. Create a small, dependency-free `harmonic_mapping.py` module derived from the two authority modules. It must expose a documented stable mapping API for a MIDI note plus f1 / anchor configuration. Preserve the 12-key natural-harmonic character and octave-aware playable frequency; do not import NaturalHarmony at runtime.
2. Integrate native note-on/off handling into the Shaper MIDI path:
   - a note-on maps to one Shaper harmonic voice / frequency using the native mapping;
   - a note-off, and note-on velocity zero, release exactly the originating voice;
   - concurrent MIDI notes must not incorrectly release each other;
   - velocity must map to bounded gain;
   - behavior works with generic keyboard ports in addition to existing Launchpad/Minilab control mappings.
3. Configuration must expose f1, anchor MIDI note, and enable/disable flag with existing style/defaults. Default standalone mode is native source enabled; `--slave` keeps consuming optional `/beacon/voice/*` broadcasts without requiring it.
4. Do not rename existing `/digital/*` OSC routes or change the current contract semantics in this task.
